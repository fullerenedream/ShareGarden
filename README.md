ShareGarden
=============

a group midterm project created at Lighthouse Labs Web Development Bootcamp
## Getting Started

1. `bundle install`
2. `shotgun -p 3000 -o 0.0.0.0`
3. Visit `http://localhost:3000/` in your browser
